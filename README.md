# Up Down Game

## Goal
1. 슬라이더 1~30이 제공됨
2. 사용자는 슬라이더를 이동하여 랜덤 타겟 번호에 근접한 수를 예상 후 HIT 버튼 터치
3. 사용자의 점수가 타겟보다 높다면 슬라이더의 MaxValue를 낮춤
4. 사용자의 점수가 타겟보다 낮다면 슬라이더의 MinValue를 높임
5. Start Over 버튼을 누르면 게임 초기화(Reset)
6. Info 버튼을 누르면 제작자의 정보가 담긴 View로 이동
